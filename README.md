# vui-jasmine-dom-matchers
[![NPM version][npm-image]][npm-url]
[![Build status][ci-image]][ci-url]

A set of Jasmine matchers for more reader friendly style specs.

## Installation

Install from NPM:
```shell
npm install vui-jasmine-dom-matchers
```

## Contributing
